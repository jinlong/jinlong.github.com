继续升级

添加更多有用的任务：

  • 使用 Autoprefixer ，CSS 中自动加前缀
  • JS 单元测试（比如：Jasmine ）
  • 自动生成图片雪碧图和 SVG 图标（比如：Grunticon ）
  • 作为文件资源服务器
  • 代码质量工具： HTML-Inspector ， CSS Lint 或者 JS Hint
  • 跨浏览器 CSS 注入
  • 帮助提交版本控制仓库，比如 GitHub
  • 添加资源版本号（清缓存）
  • 帮助部署测试和生成环境（比如：DPLOY ）
