git submodule 子模块

Git submodule 可以轻易地将别人的 git 项目挂入你目前 git 项目的任何位置

  • git submodule add

使用

需要注意 [欲放置的位置] 不能以 / 结尾（会造成修改不生效），也不能已经是现有的路径（不能顺利 Clone）。
