安卓版 Opera 14 新在哪？

发表于 2013-05-22   |   分类于 Opera   |  

相关资料：
Opera 14 for Android is out!
OUT TODAY: OPERA FOR ANDROID, FULL AND FINAL!
先唠一下 Opera 14 新功能哈，作为开发者可以直接移步这里。

经过数月的努力，全新安卓版 Opera 面世了，许多新奇特性等你发现。等不及的话，这有下载链接 Google Play 或者手机访问 m.opera.com 。

选项卡

从缩略视图到隐身标签，安卓版的选项卡非常适于浏览移动网页。把选项卡加入收藏也非常便捷，点击地址栏的星标。

快速拨号，彻底改造

如大家所愿，这版的快速拨号有文件夹了，可以自由整理，只需拖拽即可创建新文件夹。

发现，先人一步

获取最新最酷的内容，为你量身定制的发现特性。

越野模式

启用越野模式即启用了 Opera Mini 功能，当您网络不佳时，压缩网页，提高速度，省钱省流量。

下载管理

开始，暂停，恢复下载，重命名文件，您可以便利地下载保存电影，音乐和其它媒体文件。

文本自动折行

beta 测试期间，自动折行深受喜爱，正式版的来了。

可定制导航栏

设备和屏幕不同，导航栏在屏幕上方或下方，随您选择。

这仅仅是个开始

安卓版 Opera 支持姜饼（Android 2.3）及以上版本，即将支持平板。感谢所有帮我们测试 beta 版的同学，图文混排，全屏模式以及其它特性感谢您的反馈。

Google Play 下载

————–以下面向开发者————–

安卓版 Opera 14 基于 Chromium 26，UI 更符合最新的安卓设计指南。Google Play 下载 ，或者手机访问 m.opera.com 。

面向 Android 2.3 +

由于 38.5% 的安卓用户还在用姜饼，现在他们可以像更高版本的安卓用户一样，享受更好的浏览体验。

面向平板的 Opera 14 不久就会发布，我们仍在着力优化各种平板特定 UI。

全新的引擎

Opera 14 基于 Chromium 26 ，但还没用 Blink 。我们增加 / 启用了一些标准实现：

  1. <input type=color>已支持：试试这个例子
  2. 启用 WebGL 3D context ：你可以试试我们的 Odin demo，还没开始玩 WebGL ？
  3. 支持 CSS3 @supports
    《CSS3 Conditional 简介》，可以用 CSS 做特性检测。

我们处理 HTML5 audio 的方式更酷啦：如果你开始播放一首歌（例如在 SoundCloud 播放这首歌），你会看到一个“暂停按钮”出现在安卓通知栏。现在可以切换选项卡，甚至打开其它的应用，歌曲仍在后台播放，通知栏可以方便的控制音乐。

备注：有些功能暂不支持，包括自定义搜索，进入 about://flags。

我们也调整了 UA 字符串：很像 Chrome 的 UA 格式，这版在末尾加了 OPR/14.0.1074.57453 。当然，完全可以不关注这个，可以用特性检测。

新 UI 新特性

快速拨号重新设计，支持拖拽标签分组；导航栏再设计，可设置导航栏的位置；更多高级设置（比如分享，页内查找，下载等）可以看我们的用户指南 。

越野模式

特别提一下越野模式（Off-Road），启用它即通过 Opera Mini 服务加载页面，不再需要切换浏览器即可使用 Opera Mini 的数据压缩特性，现在合二为一啦~

想了解更多关于 Opera Mini 和 Off-Road 模式的优化，参见 Opera Mini 与 JavaScript 。

用户可以设置 Off-Road 模式的图片质量高低，甚至无图（图片占页面请求的大小超过 50% ）。

文本自动折行

如果你访问这页 ，看到有些文本变大啦，无需手动缩放更易读啦。设置里打开“自动折行”，不管如何放大页面，都不需要水平拖动了。

远程调试

作为开发者肯定需要用安卓版 Opera 14 调试，Windows，Mac 和 Linux 都是可以做到的，关于 Opera 远程调试可以看这篇文章 （译者注：稍后我会翻译此篇）。
